## 2. Anthropic / Claude 内容精选

**分类：News (产品/合作/投资)**

### **a) Introducing Claude Tag**
- **发布日期：** 2026-07-15
- **核心观点：** 推出全新产品“Claude Tag”，将 Claude 嵌入 Slack 成为可被 `@` 标记的团队成员，能主动参与工作流、记忆上下文并规划未来任务。这标志着 Claude Code 从辅助编码工具演变为 **主动、团队级的 AI 协作伙伴**。内部已有 65% 的产品团队代码由其生成，应用范围正从工程扩展至产品、数据和客服等领域。
- **业务意义：** 深化企业级协作场景，将 AI 能力无缝集成到现有工作平台，提升团队生产力。通过“标签化”和“委派”模式，降低使用门槛，推动 AI 在组织内部的深度采纳。
- **链接：** [https://www.anthropic.com/news/introducing-claude-tag](https://www.anthropic.com/news/introducing-claude-tag)

### **b) Agents for financial services**
- **发布日期：** 2026-07-15
- **核心观点：** 发布针对金融服务业的十个即用型代理模板，覆盖投行Pitchbook制作、KYC文件审查、月末结账等耗时任务。模板以插件形式集成在 Claude Cowork 和 Claude Code 中，并兼容 Microsoft 365 应用套件，实现跨软件无感上下文传递。强调其旗舰模型 **Claude Opus 4.7** 在 Vals AI 金融代理基准测试中领先。
- **业务意义：** 展示 Anthropic **垂直行业解决方案** 的快速落地能力。通过提供参考架构（技能+连接器+子代理），旨在缩短企业部署 AI 代理的周期，从数月缩短至数天，直接瞄准高价值商业场景。
- **链接：** [https://www.anthropic.com/news/finance-agents](https://www.anthropic.com/news/finance-agents)

### **c) Introducing Claude for Teachers**
- **发布日期：** 2026-07-14 (内容发布)，2026-07-15 (页面更新)
- **核心观点：** 为美国 K-12 教师提供免费的高级 Claude 功能访问权限，并内置教学技能库与基于证据的课程库（已映射至全美 50 州的学术标准）。该产品旨在 **弥合教育最佳实践与教师有限时间资源之间的差距**，支持差异化教学和个性化学习。
- **业务意义：** 进入教育这一关键公共领域，通过免费和专业工具塑造品牌好感度，同时培育下一代用户习惯。其理念是 **AI 支持教师而非取代教师**，反映了在敏感应用领域的负责任态度。
- **链接：** [https://www.anthropic.com/news/claude-for-teachers](https://www.anthropic.com/news/claude-for-teachers)

### **d) Anthropic commits $10 million to Canadian AI research**
- **发布日期：** 2026-07-14 (内容发布)，2026-07-15 (页面更新)
- **核心观点：** 宣布投入 1000 万加元资助加拿大 AI 研究机构，与 Amii、Mila、Vector Institute 三大顶尖研究所建立合作。同时发布基于 Anthropic 经济指数的加拿大国家简报。此举旨在支持 **有益且负责任的 AI 应用研究**，并巩固与全球 AI 研究重镇的联系。
- **业务意义：** 加强全球科研生态投资，吸引顶尖人才与基础研究合作。这既是企业社会责任，也是长期技术战略布局的一部分，尤其加拿大是强化学习等关键领域的发源地。
- **链接：** [https://www.anthropic.com/news/canadian-ai-research](https://www.anthropic.com/news/canadian-ai-research)
